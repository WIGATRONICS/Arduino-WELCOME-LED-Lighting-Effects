# Arduino WELCOME LED Lighting Effects

This project uses Arduino to create eye-catching LED lighting effects that spell out "WELCOME". Each letter is connected to a digital pin and controlled through different animation patterns.

## 🔧 Features
- All LEDs ON/OFF control
- Running light (left to right & right to left)
- Blinking effect
- Alternating pattern
- Gradual build-up (WELCOME reveal)

## 🧰 Components Required
- Arduino board (Uno, Nano, etc.)
- 7 LEDs (for W E L C O M E)
- 7 resistors (220Ω recommended)
- Jumper wires
- Breadboard

## 🔌 Pin Configuration
| Letter | Arduino Pin |
|--------|------------|
| W      | 2          |
| E      | 3          |
| L      | 4          |
| C      | 5          |
| O      | 6          |
| M      | 7          |
| E      | 8          |

## ⚙️ How It Works
Each LED represents a letter in "WELCOME". The Arduino cycles through multiple lighting effects:
- Build-up animation
- Blinking sequence
- Running lights
- Alternating pattern

## ▶️ How to Use
1. Connect LEDs to the specified pins
2. Upload the code to your Arduino
3. Power the circuit
4. Watch the animation run automatically
